# OddsEdge — Premier League Odds Comparison

Live odds comparison dashboard for English Premier League, aggregating odds from 50+ bookmakers worldwide with arbitrage detection.

## Features
- Real-time odds from 50+ bookmakers
- Arbitrage opportunity detection
- Value bet highlighting
- Mobile-responsive design
- Dark mode UI

## Tech Stack
- FastAPI backend
- Vanilla JS frontend
- The Odds API
- Render hosting

## Setup
1. Get API key from [The Odds API](https://the-odds-api.com)
2. Set `ODDS_API_KEY` environment variable
3. Run: `uvicorn main:app --reload`
